\section{Conclusions}
\bigskip
\end{center}
{\color{gray}\hrule}
\vspace{0.5cm}
\begin{multicols}{2}
    \begin{itemize}
        \item A simple two dimensional soft body simulation using spring-mass arrangement was implemented.
        \item Learned and applied concepts of Object oriented programming to write reusable and scalable code.
        \item Analysed the compute time, stability and energy flow of the soft body simulation.
    \end{itemize}
\subsection{Future Goals}
\begin{itemize}
    \item Implement different configurations of springs mass nodes that the one used in this project (in <figure>, for example) and study their dyanmics.
    \item Study collision of soft body with rigid bodies and other soft bodies by analyzing the time of collision.
    \item Three dimensional rigid body and rigid body rotation.
    \item Application to cell dynamics modelling.
    \item Finite Element Analysis method for implement soft body.
\end{itemize}
